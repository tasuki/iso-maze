## Level editor

- `e` toggle editor
- `c` reset camera
- `` ` `` twice, show maze analysis

### Move the focus

- `h` NW
- `j` SW
- `k` NE
- `l` SE
- `u` Down
- `i` Up

### Act on the focused tile

- `space`: toggle ground
- `s`: stairs (press multiple to change direction)
- `b`: bridge
- `a`: place snowman
- `z`: place hat
